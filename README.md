# Mozy's

Single-page landing site for Mozy's — charcoal chicken in Liberty Village, Toronto.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

Brand: `#000000` + `#FF9E1B` on warm bone background.
